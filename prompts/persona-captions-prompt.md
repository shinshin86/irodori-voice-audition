# ペルソナ・キャプション生成プロンプト（Claude Code / Codex 用）

`captions.json` を（再）生成したいときに、このプロンプトをそのまま渡す。
Irodori VoiceDesign は「声を言葉で説明したキャプション」から声を作るので、
**多様な声質のキャプションをばらけさせる**ほど、聞き比べで良い声に出会いやすい。

---

## プロンプト本体

次の仕様で、Irodori VoiceDesign 用のペルソナ・キャプションを **50件** 生成し、
`captions.json`（配列形式）として出力してください。

出力フォーマット（厳守）:

```json
[
  { "file": "voice_01.wav", "caption": "..." },
  { "file": "voice_02.wav", "caption": "..." }
]
```

- `file` は `voice_01.wav` 〜 `voice_50.wav`（ゼロ埋め2桁・連番）。
- `caption` は **声の説明**（読み上げる内容ではない）。日本語で、1〜2文・30〜45字程度。
- 例の書き味に寄せる: 「落ち着いた大人の女性。低めの声でゆっくり丁寧に」「明るく元気な少女。ハキハキとテンポよく」。

### 多様性の要件（できるだけ軸を散らす）
- **性別/年齢**: 少女・少年・青年・成人女性・成人男性・老年 などを混ぜる。
- **声の高さ/質感**: 高い/低い、ハスキー、澄んだ、鼻にかかった、しわがれ など。
- **話し方/速度**: ハキハキ/ゆっくり/早口/間延び/淡々 など。
- **雰囲気/感情の傾き**: 明るい・クール・優しい・強気・儚げ・妖艶・無機質 など。
- **キャラ類型**: お嬢様/委員長/ツンデレ/癒し系/ナレーター/執事/アイドル/AIアシスタント など。
- 同じような声が固まらないよう、隣接する項目はできるだけ別方向に振る。

### 禁止・注意
- 特定の実在人物・声優・キャラクターの名指し・模倣指定はしない（「〇〇っぽい声」で実在名を出さない）。
- 読み上げ文（セリフ）は含めない。あくまで「どんな声か」の説明だけ。

生成後、`captions.json` としてこのフォルダ直下に保存してください。
（件数を変えたいときは「50件」を任意の数に置き換える）
